# 邏設-final-project
- FPGA：Cyclone3 EP3C10E144C8
- software : Quartus 13.1
  
# 題目
- 閃躲遊戲，玩家可控制人物左右移動，掉落物有兩個，一個為紅色，另一個為綠色，紅色的掉落速度比綠色快兩倍，被掉落物撞到則顯示結束畫面。
